# 💰 Finances

App de controle financeiro pessoal estilo Mobills, com suporte a múltiplos usuários e workspaces (multi-tenant).

## 🧱 Stack

- **Backend:** Node.js + TypeScript + NestJS + Prisma + PostgreSQL
- **Frontend:** React + Next.js + TailwindCSS
- **Autenticação:** JWT
- **Arquitetura:** Clean Architecture + DDD-like
